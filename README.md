                                          Gmail- Clone
Building a gmail clone with the proper functional working of the exact gmail its not an UI clone.

Tech Stack Used-
      ReactJS
      Tailwind Css
      Firebase (Virtual Database)
-- Helper Library :
      Material UI
      React Icons
      React-Router-Dom
      React-Toastify
